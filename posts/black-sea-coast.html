Sample Post

January 15, 2024

This is a sample post that demonstrates how you can drop HTML files in the posts directory and they will work with the SPA system.

When you click on a post link, the SPA will first check if the post content is embedded in JavaScript. If not found there, it will try to fetch the HTML file from the posts directory.

How It Works

This hybrid approach gives you flexibility:

  • Embedded Posts: Fast loading, no server needed
  • HTML Files: Easy to edit, can be created with any HTML editor
  • Automatic Fallback: SPA tries embedded first, then fetches HTML

Creating New Posts

To create a new post, simply:

  1. Create an HTML file in the posts directory
  2. Use the same structure as this file
  3. Make sure the content is wrapped in <div id="content">
  4. Add a link to the post in your main grid

The SPA will automatically detect and load your new post!
